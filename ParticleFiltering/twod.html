This is an 800m x 800m box. The robot (green) gets a noisy distance to the nearest beacon (red), and also the name of the beacon.

It is possible for nearest beacon to be identified incorrectly. That possibility increases with increasing Distance SD. For instance the robot might be 50m from beacon 1, and 30m from beacon 2; if beacon 1 reports a distance 38m and beacon 2 reports a distance of 39m then the only information given to the robot is that it's nearest beacon is beacon 1 at a distance of 38m.

The beacons will be placed randomly everytime the system starts or the slider selecting the number of beacons changes

The orange dot is the best estimate of the robot position, which is an average over all particle positions. The 3 ellipses represent multiples 1,2, and 3 of the x and y standard deviations and give an example of how confident the particle filter is.
